# Day of AI Rwanda Certificate Verification

A simple web application for verifying the authenticity of certificates issued by Day of AI Rwanda.

## Features

- Enter a 6-digit alphanumeric certificate ID
- Client-side validation for ID format
- API integration to check certificate validity
- Direct redirect to certificate PDF upon successful verification
- Error handling with user-friendly messages
- Responsive design with modern UI

## How to Use

1. Open `index.html` in a web browser
2. Enter your certificate ID (6 alphanumeric characters)
3. Click "Verify Certificate ▶"
4. If valid, you'll be redirected to the certificate PDF
5. If invalid, an error message will appear

## Setup

This is a static website with no build process required. Simply open `randi-c-dubs.github.io/doai-cert-site` or `dayofai.org/rw-certificate` in any modern web browser.
